# Pull Request

## Forked strategy

This repository managed based on forked pull request strategy

```sh
# Fork this repository to you
$ git clone [YOUR_REPOSITORY_URL]
$ cd octodirect
$ npm install
$ npm start
# (Working...)
$ git commit [...]
$ git push origin [YOUR_REPOSITORY]
# Enroll pull-request!
```
